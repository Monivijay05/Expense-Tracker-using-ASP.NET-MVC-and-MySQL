# 💰 Expense Tracker Web Application

## 📌 Project Description
This is a web-based Expense Tracker application developed using ASP.NET MVC and MySQL.  
It helps users to manage and track their daily expenses efficiently.

---

## 🚀 Features
- User Registration & Login
- Add, Edit, Delete Expenses (CRUD)
- Category-wise Expense Management
- Dashboard with summary
- Monthly Expense Reports

---

## 🛠️ Technologies Used
- ASP.NET MVC
- C#
- MySQL
- ADO.NET
- HTML, CSS, Bootstrap
